# Workflow de vérification ESLint

## 1. Exécuter le linter
// turbo
```bash
npm run lint
```

## 2. Analyse des résultats
- Si aucune erreur : informer l'utilisateur que le code est conforme
- Si des erreurs sont détectées : les lister et proposer des corrections
